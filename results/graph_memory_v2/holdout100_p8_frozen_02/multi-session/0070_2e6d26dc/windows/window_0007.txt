The Sofia Fluorescent Immunoassay Analyzer is a bench top analyzer used for fluorescent detection of various pathogens. The system uses an ultraviolet LED source and a sensor to collect data from each test cassette. Available tests which are purchased separately include hCG, Influenza A and B, RSV, and Strep A. The analyzer may be used in two different workflow modes. The walk away mode is designed for use with a single patient sample that is placed in the Sofia and the cassette is developed and scanned before displaying the results. The read now mode is designed for use with several patient samples and the user manually develops the sample outside the analyzer. Once development is complete, the sample is placed in the Sofia and immediately scanned. Results are displayed within one minute. 

The Sofia Fluorescent Immunoassay Analyzer has an integrated printer and results may be printed, stored on the provided SD card, or sent to a laboratory information system (LIS).
